## 简介

SiteRun 是一款轻量级的本地 PHP 开发环境管理软件，支持 Windows 和 macOS。一键安装管理 Nginx、Apache、PHP、MySQL、Redis、phpMyAdmin，快速部署 WordPress 等 PHP 项目。

## 功能特性

- 🖥️ 可视化管理界面，告别命令行
- 🔄 Nginx + Apache 双引擎，每站点独立选择
- 📦 多版本 PHP / MySQL 并行，一键切换
- 🌐 每站点独立进程，互不干扰
- 🛒 内置软件商店，服务一键安装卸载
- 📊 实时监控 CPU、内存、磁盘使用率
- 🔧 内置配置编辑器，直接修改 nginx.conf、php.ini 等
- 🌙 深色 / 浅色主题，支持跟随系统

## 支持的服务

| 服务 | 说明 |
|------|------|
| Nginx | 高性能 Web 服务器 |
| Apache | 经典 Web 服务器 |
| PHP | 多版本 PHP-FPM |
| MySQL | 多版本关系型数据库 |
| Redis | 内存缓存服务 |
| phpMyAdmin | 可视化数据库管理 |

## 下载安装

前往 [Releases](https://github.com/zfan95/siterun/releases) 页面下载最新版本：

- **Windows**: `.exe` 安装包（Windows 10+）
- **macOS**: `.dmg` 安装包（macOS 12+，支持 Apple Silicon / Intel）

## 使用方式

1. 下载安装 SiteRun
2. 在软件商店安装所需服务（Nginx/Apache、PHP、MySQL 等）
3. 创建站点，选择 PHP 版本和 Web 服务器
4. 一键启动，浏览器访问即可开发
